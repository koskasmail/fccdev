#### Using bracket notation with variable value.

```sh
const objPerson = {
   first_name: "jaron",
   last_name: "lee"
};

var stName = "first_name";
console.log(objPerson[stName]);

stName = "last_name";
console.log(objPerson[stName]);
```

#### output
```sh
jaron
lee
```
