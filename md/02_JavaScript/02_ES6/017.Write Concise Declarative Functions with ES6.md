### [ Notes ]

  * ES6: abc.

#### example #1: ES5 function

```sh
const person = {
  name: "jaron",
  sayHello: function() {
    return `Hello! My name is ${this.name}.`;
  }
};

console.log(` ${person.name} `);
console.log(` ${person.sayHello()}`);
```

#### output
```sh
jaron
Hello! My name is jaron.
```
