### [ Task ]
  1. Add `["Paul", 35]` to the `beginning` of the myArray variable using `unshift()`.


### [ before ]

```sh
// Setup
const myArray = [["John", 23], ["dog", 3]];
myArray.shift();

// Only change code below this line
```

### [ solution ]

```sh
// Setup
const myArray = [["John", 23], ["dog", 3]];
console.log(myArray);

myArray.shift();

// Only change code below this line
myArray.unshift(["Paul",35]);
console.log(myArray);
```

### [ output result ]

```sh
[ [ 'John', 23 ], [ 'dog', 3 ] ]
[ [ 'Paul', 35 ], [ 'dog', 3 ] ]
```
