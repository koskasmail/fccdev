# 008.Understanding Case Sensitivity in Variables

### [ Notes ]
  * variables and function names are case sensitive.
  * example for three same and different names
    * Name
    * NAme
    * NAMe
    * NAME

```sh
var Name = 'Name';
var NAme = 'NAme';
var NAMe = 'NAMe';
var NAME = 'NAME';

console.log(Name, NAme, NAMe, NAME);
```
#### output
```sh
"Name" "NAme" "NAMe" "NAME"
```
