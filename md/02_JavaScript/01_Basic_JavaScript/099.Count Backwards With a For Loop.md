### [ before ]

```sh
Push the odd numbers from 9 through 1 to `myArray` using a `for` loop.
```

### [ solution ]

```sh
// Setup
const myArray = [];

// Only change code below this line
for (let index=9; index > 0; index -= 2) {
  myArray.push(index);
}

console.log(myArray);
```

```sh
[ 9, 7, 5, 3, 1 ]
```
