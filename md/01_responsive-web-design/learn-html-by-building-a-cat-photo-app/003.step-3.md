# 003.step-3

### [ Task ]
  1. Create a `p` element below your `h2` element and give it the following text: `See more cat photos in our gallery.`

### [ before ]

```sh
<h1>CatPhotoApp</h1>
<h2>Cat Photos</h2>
```

### [ solution ]

```sh
<h1>CatPhotoApp</h1>
<h2>Cat Photos</h2>
<p>See more cat photos in our gallery.</p>
```

### [ output result ]

<html>
  <body>
    <h1>CatPhotoApp</h1>
    <h2>Cat Photos</h2>
    <p>See more cat photos in our gallery.</p>
  </body>
</html>
