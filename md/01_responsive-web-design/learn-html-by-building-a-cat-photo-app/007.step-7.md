# 007.step-7

### [ Task ]
  1. Add an img element below the p element. At this point, no image will show up in the browser.

### [ before ]

```sh
    <main>
      <h1>CatPhotoApp</h1>
      <h2>Cat Photos</h2>
      <!-- TODO: Add link to cat photos -->
      <p> See more cat photos in our gallery. </p>
    </main>
```

### [ solution ]

```sh
    <main>
      <h1>CatPhotoApp</h1>
      <h2>Cat Photos</h2>
      <!-- TODO: Add link to cat photos -->
      <p>See more cat photos in our gallery.</p>
      <img>      
    </main>
```

### [ output result ]

<html>
  <body>
    <main>
      <h1>CatPhotoApp</h1>
      <h2>Cat Photos</h2>
      <!-- TODO: Add link to cat photos -->
      <p>See more cat photos in our gallery.</p>
      <img>      
    </main>
  </body>
</html>
