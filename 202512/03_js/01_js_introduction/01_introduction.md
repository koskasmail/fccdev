# 01_introduction

### [ Q&A ]

#### Q1: 
* Q: What role does JavaScript play in web development compared to HTML and CSS?
* A: JavaScript provides interactive functionality and dynamic behavior for the web page.

#### Q2: 
* Q: How does JavaScript typically interact with HTML and CSS on a webpage ?
* A: JavaScript interacts with the page to change content and styles dynamically.

#### Q3
* Q: Which of the following is true about the relationship between JavaScript, HTML, and CSS ?
* A: HTML and CSS are markup languages, while JavaScript is a programming language.
