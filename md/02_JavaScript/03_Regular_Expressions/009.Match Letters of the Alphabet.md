# 009.Match Letters of the Alphabet

### [ Notes ]
  * [`-`] hyphen character define a range of characters to match.
  * a to f in lowercase
     * [a-f] 

#### example #1
```sh
let testString = "big bag bIg bog"
let b_a2i_g = /b[a-i]g/g;
console.log(testString.match(b_a2i_g));
```

#### output
```sh
[ 'big', 'bag' ]
```
