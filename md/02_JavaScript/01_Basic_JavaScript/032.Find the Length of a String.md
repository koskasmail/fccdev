# 032.Find the Length of a String

### [ Notes ]
  * `.length` show you the length of a String
  * `.length` after the string variable or string literal.

#### example #1

```sh
var string1 = "welcome";
var string2 = 'back';
string2 += " " + string1 + ".";
console.log(string2 + ", length: " + string2.length);
```

#### output
```sh
"back welcome., length: 13"
```
