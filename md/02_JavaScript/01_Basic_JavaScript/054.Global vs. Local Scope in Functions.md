# 054.Global vs. Local Scope in Functions.md

### [ Notes ]
  * `local` and `global` variables can have the same `name`.

#### example #1

```sh
const varName = "jaron";

function testFunc() {
  const varName = "yifat";
  return varName;
}

console.log(testFunc());
console.log(varName);
```

#### output
```sh
"yifat"
"jaron"
```

### [ Task ]
  1. Add a local variable to `myOutfit` function to override the value of `outerWear` with the string `sweater`.

### [ before ]

```sh
// Setup
const outerWear = "T-Shirt";

function myOutfit() {
  // Only change code below this line

  // Only change code above this line
  return outerWear;
}

myOutfit();
```

### [ solution ]

```sh
// Setup
const outerWear = "T-Shirt";

function myOutfit() {
  // Only change code below this line
  var outerWear="sweater"
  // Only change code above this line
  return outerWear;
}

var output = myOutfit();
console.log(output);```

### [ output result ]

```sh
T-Shirt
```

-----

### [ Thoughts ]

  1. `local` and `global` variables can have the same `name`.
